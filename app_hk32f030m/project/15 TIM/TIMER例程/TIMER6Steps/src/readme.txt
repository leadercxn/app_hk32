@par Example Description 
本例程介绍的是TIMER6Steps的功能。
六步法基本输出模式按照以下六步，依次输出PWM波形
              -----------------------------------------------
             | Step1 | Step2 | Step3 | Step4 | Step5 | Step6 |
   ----------------------------------------------------------
  |Channel1  |   1   |   0   |   0   |   0   |   0   |   1   |
   ----------------------------------------------------------
  |Channel1N |   0   |   0   |   1   |   1   |   0   |   0   |
   ----------------------------------------------------------
  |Channel2  |   0   |   0   |   0   |   1   |   1   |   0   |
   ----------------------------------------------------------
  |Channel2N |   1   |   1   |   0   |   0   |   0   |   0   |
   ----------------------------------------------------------
  |Channel3  |   0   |   1   |   1   |   0   |   0   |   0   |
   ----------------------------------------------------------
  |Channel3N |   0   |   0   |   0   |   0   |   1   |   1   |
   ----------------------------------------------------------
利用SysTick  interrupt每10ms产生一个中断，在中断服务程序里产生TIM COM event，这样每次进COM中断服务程序从新配置TIM，依次配置让TIMER 6个通道按以上6步输出PWM波形。

例程的IO配置：
TIM1_CH1		PC6；
TIM1_CH2		PC7；
TIM1_CH3		PD3；
TIM1_CH1N	PA1；
TIM1_CH2N	PA2；
TIM1_CH3N	PA3；
TIM1_Break	PB5；

初始化
