@par Example Description 
本例程介绍的是TIMER7PWMOutputs的功能。
高级定时器有3对互补Channel和1个单一Channel，可以同时输出7个PWM波形

例程的IO配置：
TIM1_CH1		PC6；
TIM1_CH2		PC7；
TIM1_CH3		PD3；
TIM1_CH4		PD4；
TIM1_CH1N	PA1；
TIM1_CH2N	PA2；
TIM1_CH3N	PA3；
TIM1_Break	PB5；

初始化
